Getting Started

1

Get the disk image

You need a Windows 3.1 hard disk image (.img) with Castle of the Winds installed. Download one from archive.org or create one from your own files using a tool like Wine or VirtualBox.

2

Load the image

Click "Load Disk Image" and select your .img file. The emulator will start automatically. All processing happens locally in your browser — your file is never uploaded anywhere.

3

Launch the game

Once Windows 3.1 boots, navigate the Program Manager and double-click the Castle of the Winds icon (or find CW1.EXE in File Manager) to start playing.

4

Save your progress

Use the Save State button below to preserve your exact position, inventory, and dungeon state. Saved states persist in this browser until you clear site data.

Save & Load Game State

Save states capture the entire machine state (CPU, RAM, disk) — not just in-game saves. This means you can resume exactly where you left off, mid-dungeon. Download saves to back them up or play on another device.

Tips & Keyboard Shortcuts

  • Click the emulator first to capture keyboard/mouse input. Press Escape to release.
  • Ctrl+C / Ctrl+V — Use the "Fullscreen" button for a better playing experience.
  • Castle of the Winds 1 is freeware. The sequel Lifthransir's Bane is also freeware.
  • Saving in-game writes to the disk image in RAM. Use Save State to persist it across sessions.
  • Windows 3.1 tip: Alt+F4 closes windows. Double-click title bar to maximize.
  • Performance: The emulator runs in a single browser tab. Close other tabs for best speed.
